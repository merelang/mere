Mere CSV export

Pair to the SSE chat demo. SSE pushes small events to many long-lived subscribers; this pushes a large payload to one client, one row at a time, via a chunked-transfer streaming response. The Mere handler runs a while loop over 1..N and calls http_stream_write per row — no full-body allocation in the Mere heap.

generate

Cap is 1,000,000 rows. Larger requests are still valid HTTP; try curl -s 'http://localhost:8080/api/logs.csv?rows=100000' | wc -l to see the raw stream.
