仕組み

各 demo は同じ pipeline で生成されています:

  1. mere -w demo.mere で WAT (text format) を出力
  2. wat2wasm で binary .wasm に
  3. 本 HTML が fetch + WebAssembly.instantiate で読み込み + 実行
  4. Wasm の puts import を JS 側で「memory から C-string 読んで出力域に append」 として実装

つまり「Mere ソース → Wasm → ブラウザで実行」 の 4 backend feature parity を そのまま browser に持ち込んでいます。
